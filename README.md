# Product Admin — Административная панель управления товарами

Административная система для управления товарами с авторизацией, таблицей с сортировкой, поиском и пагинацией. Реализована на React + TypeScript с использованием Material-UI и FSD архитектуры.

## 🚀 Демо

🔗 **GitHub Pages:** [https://osipovfrontend.github.io/product-admin/](https://osipovfrontend.github.io/product-admin/)

## 📦 Стек технологий

| Технология | Назначение |
|------------|-----------|
| **React 18** + **TypeScript** | Фреймворк и типизация |
| **React Router v6** | Маршрутизация |
| **Zustand** | Управление состоянием |
| **Material-UI (MUI)** | UI компоненты и стилизация |
| **FSD** | Архитектура проекта |
| **Vite** | Сборка |

## ✨ Функциональность

- 🔐 **Авторизация** с выбором типа сессии (localStorage / sessionStorage)
- 📋 **Таблица товаров** с сортировкой по цене и рейтингу
- 🔍 **Поиск** с debounce (500ms)
- 📄 **Пагинация** на сервере (5/10/25 товаров на страницу)
- ➕ **Добавление** нового товара через модальное окно
- 🗑️ **Удаление** товаров (оптимистичное обновление UI)
- 🔄 **Обновление** таблицы (кнопка Refresh)
- 💾 **Сохранение настроек** сортировки и пагинации в localStorage
- 🔴 **Подсветка** товаров с рейтингом ниже 3.5
- 🔔 **Toast-уведомления** об успешных операциях

## 🚀 Запуск проекта

```bash
# Клонирование репозитория
git clone https://github.com/osipovfrontend/product-admin.git

# Переход в папку проекта
cd product-admin

# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build
